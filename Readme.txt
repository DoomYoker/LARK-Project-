
Welcome to Legend of Cornwall

Legend of Cornwall was designed as a Linux Activity to Reinforce Knowledge

Developed By: Max Smith, Kaleb Walden, and Charles Cahill

Story: A legend told throughout the land. This brave adventure awoke in a cave to find himself thrown against the evils daring to be charted. 
This mythical quest travels across a variety of locations, meeting friendly faces and unwelcoming figures. 
But the greatest foe lies at the end of the crossroads.

Instruction: Type into the command line ". menu" to begin the story. Make sure you read everything, as it will help you progress. Currently everything has a number for the vim files specifically. Make sure to only use a vim file once and to use them in the order they appear.
